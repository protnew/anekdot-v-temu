# Anekdot v Temu - Joke in Context

An AI-powered application that delivers contextually relevant jokes based on real-time conversation analysis. Listens through the microphone, recognizes speech, identifies the conversation topic using NLP, and surfaces matching jokes from a curated database.

## Why This Project Exists

Humor is a universal social tool. A well-timed joke can break the ice at a party, defuse tension in a meeting, or make a presentation memorable. But not everyone has a mental library of jokes for every situation. This app uses AI to match conversation context with relevant humor in real-time.

## Current State

Early development in Python. Core speech recognition pipeline is functional. NLP topic extraction and joke matching are in progress.

## Vision

A real-time humor assistant that works seamlessly in any social context: accurate NLP topic extraction, a comprehensive categorized joke database, and a discreet UI that surfaces the perfect joke at the perfect moment without disrupting the conversation.
