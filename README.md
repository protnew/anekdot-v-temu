# Anekdot v Temu (Joke in Context)

An AI-powered application that delivers contextually relevant jokes based on real-time conversation analysis. Listens through the device microphone, recognizes speech, identifies the conversation topic, and surfaces matching jokes from a curated database.

## How It Works

Audio is captured through the microphone and transcribed using speech recognition. The transcribed text is analyzed with NLP to identify conversation topics, sentiment, and key themes. Based on this analysis, the app searches a curated joke database to find the best match and displays it on screen.

The idea is simple: humor is a social lubricant, but not everyone has a mental library of jokes for every situation. This app uses AI to match conversation context with relevant humor in real-time.

## Current Status

Early development. Core speech recognition pipeline is functional. NLP topic extraction and joke matching are in progress.

## Tech Stack

- **Python** — application logic and NLP pipeline
- **Speech Recognition** — real-time audio transcription
- **NLP** — topic extraction and context analysis
