# Anekdot v Temu - Joke in Context

An AI-powered application that delivers contextually relevant jokes based on real-time conversation analysis. Listens through the device microphone, recognizes speech, identifies the conversation topic using NLP, and surfaces matching jokes from a curated database.

## The Idea

Humor is a universal social tool. A well-timed joke can break the ice at a party, defuse tension in a meeting, or make a presentation memorable. But not everyone has a mental library of jokes ready for every situation. This app bridges that gap using AI to match conversation context with relevant humor in real-time.

The concept is straightforward: the app runs in the background, captures audio, transcribes speech, analyzes the topic and sentiment of the conversation, searches a curated joke database for the best match, and displays it as a discreet notification. The user gets a perfectly timed punchline to drop into the conversation.

## How AI Coding Tools Help

This project is a natural fit for AI-assisted development. The NLP pipeline for topic extraction, the joke matching algorithm, and the speech recognition integration all benefit from AI pair programming. With Codex access, the plan is to improve the NLP topic extraction accuracy, expand the joke database with automated categorization, and build a proper UI for joke browsing.

## Current State

Early development in Python. Core speech recognition pipeline is functional. NLP topic extraction and joke matching are in progress. The curated joke database is growing.

## Tech Stack

- Python - application logic and NLP pipeline
- Speech Recognition - real-time audio transcription
- NLP - topic extraction and context analysis
