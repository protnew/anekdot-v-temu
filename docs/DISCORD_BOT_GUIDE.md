# 🤖 Discord Бот «Анекдот в Тему» — Бизнес-сценарий

## Что это такое

Discord бот, который подбирает шутки по контексту разговора прямо в Discord сервере. 200K шуток, 10 языков, AI-подбор.

## Как это работает (для пользователя)

```
Сервер Discord → канал #general
──────────────────────────────
Пользователь: /joke
Бот: 😂 [случайная шутка из 200K] + кнопки [🔀 Ещё] [⭐ Оценить] [🗑 Удалить]

Пользователь: /joke тема: программист
Бот: 😂 [шутка про программистов из AI-подбора]

Пользователь: /search запрос: работа
Бот: 🔍 [топ-3 шутки про работу]

Пользователь: /top
Бот: 🏆 [топ-5 шуток по рейтингу]

Пользователь: /cat категория: айти
Бот: 😂 [случайная шутка из категории "айти"]

Пользователь: /categories
Бот: 📂 [список всех 132 категорий]

Пользователь: /stats
Бот: 📊 [200000 шуток, 132 категории, 10 языков, v3.6.0]
```

## 6 slash-команд

| Команда | Описание | Пример |
|---------|----------|--------|
| `/joke` | Случайная шутка или по теме | `/joke тема: коты` |
| `/search` | Поиск шуток по тексту | `/search запрос: начальник` |
| `/top` | Топ-5 шуток по рейтингу | `/top` |
| `/cat` | Шутка из конкретной категории | `/cat категория: айти` |
| `/categories` | Список всех категорий | `/categories` |
| `/stats` | Статистика базы | `/stats` |

## Кнопки под каждой шуткой

- 🔀 **Ещё** — следующая шутка (из той же категории/темы)
- ⭐ **Оценить** — поставить +1 к рейтингу
- 🗑 **Удалить** — убрать сообщение бота

## Бизнес-сценарий — зачем это нужно

### Сценарий 1: Развлечение комьюнити (бесплатно)
- Discord серверы геймеров, стримеров, IT-комьюнити
- Бот повышает активность в чате, создаёт атмосферу
- Вирилка: пользователи делятся шутками, приглашают бота на свои серверы

### Сценарий 2: Инструмент стримера
- Стример на Twitch/YouTube → Discord его комьюнити
- `/joke тема: стрим` → шутка про стриминг в чат
- Разрядка атмосферы, интерактив с чатом

### Сценарий 3: B2B — корпоративные Discord серверы
- IT-компании используют Discord для работы
- `/joke тема: дедлайн` → снимает напряжение в команде
- Team building, корпоративная культура

### Сценарий 4: Монетизация (дальше)
- Бесплатный бот: 20 шуток/день на сервер
- Премиум бот: безлимит + AI-генерация + мемы → 490₽/мес за сервер
- B2B: кастомная база шуток компании → 4900₽/мес

## Как получить токен (5 минут)

### Пошаговая инструкция:

1. **Открой** https://discord.com/developers/applications
2. **Нажми** «New Application» → назови «Анекдот в Тему»
3. **Слева** → «Bot» → «Add Bot» → подтверди
4. **На странице Bot:**
   - Нажми «Reset Token» → скопируй токен (виден ОДИН раз!)
   - Включи «Message Content Intent» (пониже)
   - Включи «Server Members Intent»
5. **Слева** → «OAuth2» → «URL Generator»:
   - Scopes: `bot`, `applications.commands`
   - Bot Permissions: `Send Messages`, `Use Slash Commands`, `Embed Links`, `Read Message History`
   - Скопируй ссылку и открой в браузере → выбери сервер → «Authorize»

### Запуск:
```bash
export DISCORD_BOT_TOKEN="ваш_токен_здесь"
export API_BASE="http://ваш-vps-ip:8000"
pip install -r bot/discord_requirements.txt
python bot/discord_bot.py
```

## Архитектура

```
Discord сервер
    ↕ (WebSocket)
discord_bot.py (244 строки)
    ↕ (HTTP REST)
main.py API (v3.6.0, 200K шуток)
    ↕
jokes_db.json (60MB, 200K шуток)
```

Бот — тонкий клиент. Вся логика (поиск, TF-IDF, подбор) — в API на VPS.

## Масштаб

- 1 VPS instance API → обслуживает 100+ Discord серверов
- Бот добавляется на сервер за 30 секунд (линк)
- Каждая команда = 1 API запрос → <50ms latency
