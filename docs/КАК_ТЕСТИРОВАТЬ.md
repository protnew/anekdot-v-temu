# 🧪 Как тестировать «Анекдот в тему»

## Шаг 1. Установка (один раз)

### Вариант A: С нуля
```cmd
cd "C:\Сделать\Чейчер SCRUM\ии проекты\5806 Приложение анекдот в тему"
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### Вариант B: Быстрый (bat сам всё сделает)
Просто запусти `start.bat` → выбери [1] → он создаст venv и установит пакеты.

> **Внимание:** `pip install` может занять 2-5 минут (scikit-learn ~30MB). Это нормально.

---

## Шаг 2. Запуск

Запусти `start.bat` и выбери режим:

| # | Режим | Что делает | Для чего |
|---|-------|-----------|----------|
| 1 | **Веб** | Только веб-сервер | Проверить что API работает |
| 2 | **Голос** | Сервер + микрофон | Проверить распознавание речи |
| 3 | **Полный** | Сервер + микрофон + overlay | Полный цикл |

---

## Шаг 3. Проверка API (обязательная)

Открой в браузере: **http://localhost:8000**

### Что должно быть на главной:
- Тёмная тема
- Кнопка «Случайная шутка» — нажми, должна появиться шутка
- Поле поиска — введи «начальник» → должны быть шутки про работу
- Категории слева — нажми любую → должны быть шутки этой категории

### Если не работает:
1. Проверь что в окне cmd нет красных ошибок
2. Попробуй http://localhost:8000/api/stats — должна быть JSON-строка
3. Если «Connection refused» — сервер не стартанул, смотри ошибки в cmd

---

## Шаг 4. Проверка логов

Открой: **http://localhost:8000/logs**

- Автоматически дёрнет API и покажет результат
- Кнопка **«📋 Копировать всё»** — копирует все логи в буфер
- Фильтры: Всё / Шутки / Ошибки / Голос
- Статус вверху: API/База/Whisper — зелёные если ок

---

## Шаг 5. Проверка голоса (опционально)

> Нужен микрофон и установленный PyAudio (`pip install pyaudio`)

1. Запусти `start.bat` → выбери [2] или [3]
2. Скажи что-нибудь в микрофон (по-русски)
3. Через 15-20 секунд в логах или overlay должна появиться шутка по теме

### Если голос не работает:
- «No module named speech_recognition» → `pip install SpeechRecognition`
- «No module named pyaudio» → `pip install pyaudio` (на Windows может потребоваться `pip install pipwin && pipwin install pyaudio`)
- «No Default Input Device» → подключи микрофон
- Whisper модель скачается автоматически при первом запуске (~140MB)

---

## Шаг 6. Что проверять

### ✅ Обязательные проверки:
1. **http://localhost:8000** — главная загружается
2. Кнопка «Случайная шутка» — работает
3. Поиск «программист» — есть результаты
4. Категория «работа» — есть шутки
5. **http://localhost:8000/logs** — открывается, кнопка копирования работает
6. Оценка ★★★★★ — можно поставить

### ⚠️ Опциональные проверки:
7. Голосовой режим — микрофон → шутка
8. Overlay — всплывающее окно в углу
9. Алиса навык (только если есть Яндекс.Станция)

---

## Возможные ошибки и решения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `'ние' is not recognized` | Старый bat с emoji | Используй `start.bat` (без emoji) |
| `No module named sklearn` | Не установлен venv | Запусти `start.bat` — сам установит |
| `No module named speech_recognition` | Нет пакета | `pip install SpeechRecognition` |
| `Port 8000 already in use` | Другой процесс на порту | Закрой его или убей через Task Manager |
| `Connection refused` | Сервер не стартанул | Смотри ошибки в cmd |

---
