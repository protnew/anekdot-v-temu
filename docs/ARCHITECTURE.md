# 🏗️ АРХИТЕКТУРА «Анекдот в Тему» v3.1

## Как это работает — объяснение для человека

### Простыми словами

Приложение делает одну вещь: **ты пишешь текст (контекст ситуации) → оно подбирает подходящий анекдот**.

Например: ты написал «работа начальник» → получаешь анекдоты про работу.

```

        │  ФРОНТЕНД      │  Это красивая тёмная страничка в браузере

        └───────┬────────┘
```

### Пошагово: что происходит когда ты вводишь текст

**Шаг 1.** Ты вводишь «работа начальник» на сайте

**Шаг 2.** Бэкенд берёт твой текст и превращает в числа:
- Слово «работа» → число 0.8 (часто встречается в шутках про работу)
- Слово «начальник» → число 0.6
- Это делает TF-IDF (специальная математика из sklearn)

**Шаг 3.** Сравнивает твой вектор чисел со всеми 4782 шутками
- У каждой шутки ТОЖЕ есть свой вектор (подготовлен заранее при старте)
- cosine_similarity — мера «похожести» двух наборов чисел
- Шутки с наибольшей похожестью — кандидаты

**Шаг 4.** Keyword boosting (доп. проверка):
- Слово «работа» → попадает в категорию «работа»
- Шутки из категории «работа» получают ×1.5 к рейтингу
- Шутки из других подходящих категорий — ×1.3

**Шаг 5.** Возвращает топ-3 (или сколько ты просил) шуток с оценкой похожести

### API — 29 адресов (endpoints)

Основных 4, остальные — доп. фичи:

**Главные:**
| Метод | URL | Что делает |
|-------|-----|-----------|
| POST | `/api/jokes/context` | **Главный!** Вводишь текст → получаешь подходящие шутки |
| POST | `/api/jokes/generate` | Генерирует НОВУЮ шутку через GPT (или шаблон) |
| GET | `/api/jokes/search?q=кот` | Поиск по ключевым словам |
| GET | `/api/joke/random` | Случайный анекдот |

**Каталог:**
| Метод | URL | Что делает |
|-------|-----|-----------|
| GET | `/api/categories` | Список категорий |
| GET | `/api/jokes?category=работа` | Шутки одной категории |

**Пользовательские:**
| Метод | URL | Что делает |
|-------|-----|-----------|
| POST | `/api/favorites` | В избранное (per-user!) |
| POST | `/api/rate` | Оценить 1-5 |
| POST | `/api/user-jokes` | Добавить свою шутку |

### Стек (на чём написано)

| Что | Технология | Почему |
|-----|-----------|--------|
| Сервер | FastAPI + Uvicorn | Быстрый, асинхронный |
| Поиск | scikit-learn TF-IDF | Не нужен GPU, работает на любом компе |
| Генерация | OpenAI GPT-4o-mini | Дёшево (~15 копеек за шутку) |
| Данные | JSON + SQLite | Просто, без внешних баз |
| Сайт | Vanilla JS | Грузится мгновенно |
