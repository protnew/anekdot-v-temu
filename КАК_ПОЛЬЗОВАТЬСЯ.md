# 😂 Анекдот в тему — Как пользоваться

## Что это

Приложение слушает разговор через микрофон, распознаёт речь и **автоматически подбирает шутки в тему**. Работает оффлайн (модель Whisper уже скачана).

## Запуск

**Windows:** двойной клик на `start.bat` → выбираешь режим.

**Linux/Mac:**
```bash
python launcher.py full      # всё сразу
python launcher.py voice     # бэкенд + микрофон
python launcher.py server    # только веб
```

## 3 режима

| Режим | Что делает | Для чего |
|-------|-----------|----------|
| **1. Сервер** | Запускает только бэкенд. Открой `http://localhost:8000` | Посмотреть веб-интерфейс, поискать шутки руками |
| **2. Голос** | Бэкенд + слушает микрофон. Шутки появляются в консоли | Основной режим — говоришь, шутки в тему прилетают |
| **3. Полный** | Бэкенд + микрофон + полупрозрачное окно поверх всех окон | Шутки на экране во время разговора (нужен десктоп) |

## Как работает голосовой режим

```
Ты говоришь в микрофон
        ↓
Whisper распознает речь (оффлайн, ~1 сек)
        ↓
Накапливается контекст последних 45 секунд
        ↓
Каждые 20 секунд отправляется на сервер
        ↓
TF-IDF поиск по 4782 анекдотам
        ↓
😂 Шутка на экране
```

## Примеры

Скажешь:
- «Начальник опять орал» → шутка про начальника
- «Жена сказала что я мало зарабатываю» → шутка про жену
- «Программист написал код» → шутка про программистов

## Остановка

`Ctrl+C` в консоли — останавливает всё.

## Если не работает микрофон

1. Проверь что микрофон включён в настройках ОС
2. Windows: Settings → Privacy → Microphone → Allow
3. Linux: `sudo apt install portaudio19-dev`

## Если Whisper не распознаёт

По умолчанию стоит модель `base` (быстрая, 150 MB). Можно переключить на `small` (точнее, 500 MB):

```bash
WHISPER_MODEL=small python launcher.py voice
```

## Веб-интерфейс

`http://localhost:8000` — 5 вкладок:
- 🔥 Популярные
- 🎲 Случайная
- 🔍 Поиск (введи тему — получишь шутки)
- ⭐ Избранное
- ➕ Добавить свою
